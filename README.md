# Crypto Sentiment vs Trader Performance

## 📌 Overview
This project analyzes how Bitcoin market sentiment (Fear/Greed) impacts trader behavior and profitability.

## 📊 Features
- Sentiment vs PnL analysis
- Win rate comparison
- Leverage risk insights
- Hourly trading patterns
- ML model to predict trade success

## 🚀 Tech Stack
- Python
- Pandas, NumPy
- Seaborn, Matplotlib
- Scikit-learn
- Streamlit

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
